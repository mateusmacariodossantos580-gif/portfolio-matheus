Sobre o Projeto

Este projeto é uma calculadora funcional desenvolvida utilizando HTML, CSS e JavaScript. O objetivo foi criar um layout limpo, estiloso e totalmente responsivo, além de treinar lógica de programação manipulando operações matemáticas básicas.

Código Usado

Aqui está a lógica JavaScript usada para fazer a calculadora funcionar:

// Adiciona caracteres ao visor
function add(value) {
  document.getElementById('display').value += value;
}

// Limpa o visor
function clearDisplay() {
  document.getElementById('display').value = '';
}

// Calcula
function calc() {
  try {
    let result = eval(document.getElementById('display').value);
    document.getElementById('display').value = result;
  } catch {
    document.getElementById('display').value = 'Erro';
  }
}
